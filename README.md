# Portfolio Profesional - Azael Reyes Martel

Bienvenido al código fuente de mi portfolio personal. Este proyecto ha sido diseñado y desarrollado desde cero para demostrar competencias avanzadas en estructuración de interfaces web, diseño responsive y arquitectura CSS.

## 🚀 Sobre el Proyecto

Soy un Técnico Superior en ASIR y estoy finalizando mi formación de DAW. Mi enfoque hacia el desarrollo web no se basa en simplemente escribir código visual, sino en entender la estructura y el rendimiento detrás de cada caja. Por ello, este portfolio evita el uso de plantillas prefabricadas y que el código sea fácil de entender y para el usuario sea una navegación óptima y fluida. Además, incluyo documentación detallada del proceso de desarrollo del diseño, aunque sé que es básico, pero cumple su propósito y se puede seguir mejorando con el tiempo.

### Características Principales:
- **Desarrollo "Vanilla":** construido sin frameworks CSS pesados. Se utiliza CSS Grid, Flexbox y el Modelo de Cajas para demostrar los conocimientos de Diseño de Interfaces Web.
- **Arquitectura de Variables:** uso de CSS Custom Properties para un mantenimiento escalable.
- **HTML Semántico:** estructura pensada para lectores de pantalla y buen SEO (etiquetas `<main>`, `<section>`, `<nav>`, `<header>`).
- **Diseño Responsive:** adaptado a dispositivos móviles mediante Media Queries.

## 🛠️ Stack Tecnológico

- **HTML5** Semántico
- **CSS3** (Variables, Flexbox, Transiciones)
- **Vite** (Como herramienta de *bundling* y servidor de desarrollo eficiente)

## ⚙️ Instalación y Despliegue en Local

Si deseas ejecutar este proyecto en tu propia máquina para revisar el código:

1. Clona el repositorio
   ```bash
   git clone https://github.com/AzaelReyesMartel-1ASIR/portfolio-web.git 
   ```

2. Accede a la carpeta
    ```bash
    cd portfolio-web
    ```

3. Instalar las dependencias del desarrollo
    ```bash
    npm install
    ```

4. Levantar el servidor en local
    ```bash
    npm run dev
    ```
